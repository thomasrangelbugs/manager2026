# Manager 2026

Manager 2026 é um jogo web de gerenciamento de futebol desenvolvido para navegador. O projeto simula a experiência de comandar um time, tomar decisões estratégicas, acompanhar partidas, gerenciar elenco e evoluir dentro de uma jornada esportiva.

## Funcionalidades

* Jogo de gerenciamento de futebol pelo navegador
* Escolha e controle de time
* Gestão de elenco
* Simulação de partidas
* Sistema de competições
* Evolução da equipe durante a temporada
* Interface inspirada em jogos de manager esportivo
* Layout responsivo para PC e celular
* Experiência visual voltada para futebol
* Publicação online via Netlify

## Objetivo

O objetivo do projeto é criar uma experiência simples e divertida de gerenciamento de futebol, explorando lógica de jogos, tomada de decisão, organização de dados, simulação de partidas e interface interativa.

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Netlify
