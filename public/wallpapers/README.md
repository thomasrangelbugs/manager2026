# Wallpapers da tela inicial

Coloque aqui imagens `.jpg` ou `.webp` (ex.: `01.jpg`, `02.jpg`) e adicione entradas em `src/data/splashWallpapers.ts`:

```ts
{ id: 'local-01', src: '/wallpapers/01.jpg', label: 'Seu texto' },
```

Recomendado: 1920×1080 ou maior, paisagem, jogadores/estádio do Brasileirão.
