# Manager 2026

> Jogo web de gerenciamento de futebol para navegador.

## Sobre o projeto

Simulador de gerenciamento de futebol desenvolvido para navegador, inspirado em managers esportivos classicos. Permite escolher e comandar um time, gerenciar elenco, simular partidas, participar de competicoes e evoluir a equipe ao longo da temporada. Interface visual voltada para futebol com suporte a PC e celular.

## Funcionalidades principais

- Escolha e controle de time
- Gestao de elenco e simulacao de partidas
- Sistema de competicoes e evolucao da equipe
- Interface inspirada em managers esportivos
- Layout responsivo para desktop e mobile

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Netlify

## Como executar

Abra index.html no navegador.
